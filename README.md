# Fixed Ops Tracker

Dealer Operating Control Service — Fixed Ops Daily Tracker.

## GitHub / Vercel setup

Upload these files to your GitHub repository:

- `index.html`
- `package.json`
- `tailwind.config.js`
- `postcss.config.js`
- `src/App.jsx`
- `src/main.jsx`
- `src/style.css`

Vercel should use:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
